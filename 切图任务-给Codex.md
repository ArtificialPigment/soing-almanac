## 目标产物

请产出两套，命名一律用上面的 `<id>`：

### A. 干净的手绘插画（最重要）
- 路径：`site/assets/illustrations/<id>.png`（**透明背景 PNG**）。
- 内容：**只保留这杯酒**（杯子 + 酒液 + 装饰物，如柠檬片/樱桃/薄荷/吸管/橙皮等），
  **去掉牛皮纸背景**、去掉所有文字、边框、装订环、桌面阴影。
- 画幅：统一放到**同一画布比例**（建议 **4:5 竖版**或正方形），主体**居中**，四周留**统一边距**（约 8–10%）。
- 分辨率：长边 ≥ **1200px**，线条锐利（可用 AI 超分，如 Real-ESRGAN，但**不要把水彩抹成塑料感**）。
- 风格统一：20 张放在一起要像**同一套**（尺寸、留白、明度、白平衡一致）。

> 备选/补充：再导出一套**牛皮纸底**版本 `site/assets/illustrations/<id>.jpg`，
> 把各张牛皮纸**白平衡/亮度归一**到同一色（消除现有「深浅不一」问题），作为不便用透明 PNG 时的回退。

### B. 更干净的正视卡片
- 路径：`site/assets/cards/<id>.jpg`（约 820–1000px 宽）。
- 要求：透视校正更准，**裁到印刷双线边框以内**（去掉顶部金属环/麻绳和外缘杂边），
  各张**长宽比一致**、边缘干净。

## 推荐技术路线（可自行优化）

你应该可以联网装包（`pip install opencv-python numpy pillow rembg onnxruntime scikit-image`）。

1. **卡片检测 + 透视校正**（产物 B，也用于 A 的输入）
   - 用 OpenCV：转灰度/HSV → 阈值出「暖色牛皮纸」区域（高明度、低蓝、暖色：`R-B` 较大）；
     **形态学开运算**去掉左上角卡背扇形的细碎连通；`findContours` 取最大四边形
     （`approxPolyDP`）；`getPerspectiveTransform` + `warpPerspective` 正视化。
   - 再**向内裁 ~3–4%** 到双线边框以内，去掉装订环与外缘。

2. **插画分割/去背**（产物 A，核心难点）
   - 牛皮纸底色相对均匀：构建「**非牛皮纸**」掩膜（与局部背景在色相/明度上差异超阈值的像素）
     → 形态学闭运算 → 取**最大连通域**（即这杯酒，排除右侧文字列与边框）→ 羽化边缘成 alpha。
   - 注意水彩**半透明**，玻璃杯内部会透出牛皮纸色：可对「杯型轮廓内部」做**凸包/填充**保住整杯，
     避免把杯身键成透明。可结合 `rembg`（U²-Net）做前景兜底，再与上面的掩膜取并集/相交择优。
   - 自动 trim 到主体外接框 → 贴到统一画布居中 → 导出 PNG（带 alpha）。
   - **务必排除右侧文字**：插画只在卡片**左半**（`x < 0.5W`），可先限定区域再分割。

3. **统一化**：所有 PNG 统一画布与边距；牛皮纸底版本统一白平衡（按每张背景采样的中位色做增益校正）。

4. **质检图**：拼一张 5×4 的 contact sheet（透明 PNG 叠在中性灰上）方便人工核验。

## 验收标准（逐张自检）
- [ ] 插画里**没有任何文字、边框、装订环、桌面**，只有这杯酒 + 装饰。
- [ ] 透明 PNG 在深色背景上边缘干净、无白边/牛皮纸残留；杯子是完整的（杯身没被误抠空）。
- [ ] 20 张主体**居中**、画幅/留白一致，放一起成套；明度/白平衡统一。
- [ ] 卡片版裁到边框以内，无金属环/杂边，长宽比一致。
- [ ] 长边分辨率达标、线条锐利、无过度涂抹。

## 与网站对接（很重要，别漏）
网站当前这样引用素材（见 `work/merge.js` 生成的 `site/js/data.js`，字段 `illu` / `card`）：
- `card` → `assets/cards/<id>.jpg`
- `illu` → `assets/illustrations/<id>.jpg`

如果插画改成 **PNG**，二选一：
1. 仍另存一份 `.jpg` 牛皮纸底版给现引用用；或
2. 改 `work/merge.js` 里 `illu` 为 `assets/illustrations/${id}.png`，然后 `node work/merge.js` 重新生成 `site/js/data.js`。

另注意：插画用在「酒单卡片(menu tile)」是 `object-fit:cover` 铺满图区；
若换成**透明 PNG**，请把 `site/css/style.css` 里 `.tile-art img` 改为 `object-fit:contain` 并给图区一个底色，
让水彩酒「浮」在卡面上（会更好看）。`.hero-motifs img`、详情页同理可微调。

完成后请：`node work/merge.js`（如改了引用）→ 本地 `python -m http.server` 起 `site/` 自查 →
`git add -A && git commit && git push`，并把 `site/` 用 `git subtree push --prefix site origin gh-pages`
推到 Pages（仓库：`ArtificialPigment/soing-almanac`，线上 `https://artificialpigment.github.io/soing-almanac/`）。
