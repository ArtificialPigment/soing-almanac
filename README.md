# 醺谱 · SOING ALMANAC

> 二十张手绘鸡尾酒卡，被重新装订成一册可以**抽签**、可以**翻阅**的午夜酒谱。

把二十张实体手绘卡片（品牌 **SOING**）逐张校正、提炼，连同原料、配比、做法与背后的故事，
做成了一个有调性的鸡尾酒网站。

**🍸 在线访问：** https://artificialpigment.github.io/soing-almanac/

---

## 快速开始

网站是**纯静态**的（HTML / CSS / 原生 JS，无需构建）。两种方式：

**方式一：本地服务器（推荐，字体/图片最稳）**
```bash
cd site
python -m http.server 8842
# 浏览器打开 http://127.0.0.1:8842/
```

**方式二：直接打开**
直接双击 `site/index.html` 也能运行（需联网以加载 Google Fonts）。

## 三大核心功能

| 功能 | 路径 | 说明 |
| --- | --- | --- |
| **盲盒抽签** | `#/blindbox` | 轻触牌堆，卡片从卡册中被抽出、翻面亮相（3D 翻牌动画 + 烫金光晕），可「再抽一张」或「查看这一杯」。深链 `#/blindbox/auto` 会自动抽一张。 |
| **酒单菜单** | `#/menu` | 二十张手绘卡的网格图鉴，可按**基酒**筛选（金酒 / 威士忌 / 朗姆 / 伏特加 / 龙舌兰 / 白兰地 / 餐前苦酒）。 |
| **详情展示** | `#/cocktail/<id>` | 手绘原卡 + 名称 + 故事 + 风味/杯型/酒精度等信息 + **材料**（带配比）+ **逐步做法** + 调酒师笔记，并可前后翻页。 |

## 设计

- **调性**：午夜酒馆 / 暗房酒志 —— 深咖墨色背景、胶片颗粒、聚光灯晕染，牛皮纸卡片如展品般发光，黄铜金描边与装饰角线呼应卡面边框。
- **字体**：`Cormorant Garamond`（拉丁衬线）+ `Pinyon Script`（手写花体，呼应卡面）+ `Noto Serif SC`（中文）。
- **动效**：入场渐起、抽牌翻面、卡片悬浮、票根滚动等，均尊重 `prefers-reduced-motion`。
- 响应式，移动端单独适配。

## 目录结构

```
cocktail/
├── site/                      # ← 网站本体（部署只需这个目录）
│   ├── index.html
│   ├── css/style.css
│   ├── js/
│   │   ├── app.js             # 路由 + 各视图 + 动画
│   │   └── data.js            # 生成产物：window.ALMANAC（20 杯全部数据）
│   ├── data/
│   │   ├── recipes.base.json  # 据卡面忠实转录的配方（名称/原料/做法）
│   │   └── packs.json         # 故事/风味/杯型/笔记/配图提示词（AI 撰写并核校）
│   └── assets/
│       ├── cards/             # 20 张校正去背后的手绘原卡（JPG）
│       ├── illustrations/     # 20 张裁切出的手绘插画（JPG）
│       └── generated/         # 放 AI 生成的成品图（<id>.jpg，详情页自动显示）
├── 图片提示词.md               # ← 用 ChatGPT/DALL·E 生图的提示词手册
├── work/                      # 处理脚本与中间产物（非部署所需）
│   ├── deskew.py              # 从照片中检测卡片并透视校正
│   ├── extract.py            # 切出网页用的卡片与插画
│   ├── merge.js               # 合并 base + packs → site/js/data.js
│   └── shots/                 # 验收截图
└── 20260626-*.jpg             # 20 张原始照片
```

## 内容是怎么来的

1. **提取**：20 张照片里，卡片被拍得倾斜且占比不一。`work/deskew.py` 用纯 Pillow
   按「暖色牛皮纸」特征把卡片从深色桌面中识别出来，并做透视校正，得到干净竖版卡片。
2. **截取手绘**：`work/extract.py` 在校正后的卡片上自动定位水彩插画区域（在文字列之前的留白处截断），切出 `illustrations/`。
3. **提炼文字**：逐张读取卡面，忠实转录名称、原料、做法到 `recipes.base.json`。
4. **补全故事**：为每杯撰写背景故事、风味、杯型、装饰、调酒师笔记与配图提示词，并经**史实核查**润色，存入 `packs.json`。
5. **合并**：`node work/merge.js` 生成 `site/js/data.js`。

> 改了 `recipes.base.json` 或 `packs.json` 后，重新跑 `node work/merge.js` 即可更新网站数据。

## 让内容更图文并茂（可选）

打开 **`图片提示词.md`**，里面有：
- 20 杯**成品 / 原料静物**提示词（每杯一条，含正确酒杯与关键原料）；
- 8 种**酒杯图鉴**与 20 种**关键原料/基酒**提示词；
- 站点氛围图提示词。

用生图工具生成后，把图命名为对应 `id.jpg` 放进 `site/assets/generated/`，
详情页就会在手绘卡下方自动出现「成品参考图」。
