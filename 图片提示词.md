# 醺谱 · 配图提示词手册

> 用这些提示词在 **ChatGPT / DALL·E / Midjourney** 等工具生成配图，丰富网站的图文表现。

## 使用方法

1. 复制下方任一英文提示词，发给生图工具。
2. 想要**成品参考图自动出现在「详情页」**：把生成的图命名为对应的 `id.jpg`（见每条标题括号），放进 `site/assets/generated/` 文件夹即可——网站会自动在该酒的手绘卡下方显示；没有图时则自动隐藏，互不影响。
3. 建议统一风格：**复古手绘水彩 + 植物学插画**，米色/牛皮纸背景，无文字水印，竖构图，使整站调性一致。
